osutools resource list

This is a list of all 3rd party tools related to the rhythm game osu!

If you would like your work added to the list, please make a pull request at the github repo.

  • USA State Leaderboard

    A self-maintined leaderboard for all 50 states.

  • Custom Leaderboards

    Leaderboards for many different skillsets or groups of people.

  • Skill Breakdown

    Shows which skills you are better or worse at compared to your peers.

  • Stream Companion
  • Real Time PP Display
  • Hawku

    The "original" tablet driver. Unmaintained, but still very popular and stable.

  • Devocub

    Fork of the Hawku drivers with some added features and is still being worked on.

  • Open Tablet Driver

    Cross platform tablet drivers with no admin access required, peppy has implemented a version of this into lazer.

  • FunOrange osu! Trainer
